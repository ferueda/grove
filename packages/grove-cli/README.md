# @ferueda/grove-cli

> A fast, secure pool of reusable git worktrees with durable branch-aware leases

CLI for Grove — lease-first commands (`acquire`, `list`, `inspect`, `release`, `repair`, `destroy`) with stable `--json` envelopes.

See the [main documentation](https://github.com/ferueda/grove#the-grove-cli) for installation and command reference.
